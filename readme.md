# 🍽️ Restaurant - Site Vitrine

Bienvenue sur le dépôt GitHub du projet **Restaurant**.

Ce projet a été réalisé dans le cadre de ma formation **BTS SIO option SLAM**, pour apprendre à concevoir un site web vitrine avec réservation en ligne.

---

## 📋 Présentation du projet

Le projet **Restaurant** est un site vitrine pour un restaurant fictif, présentant :
- L’établissement (description, photos)
- La carte des menus
- Les horaires d'ouverture
- Un formulaire de réservation en ligne

L'objectif est de proposer un site simple, clair et responsive (adapté aux mobiles et tablettes).

---

## 🚀 Technologies utilisées

- **HTML5** : Structure des pages
- **CSS3** + **Bootstrap** : Mise en forme responsive
- **JavaScript** : Validation des formulaires
- **PHP** : Traitement serveur des réservations
- **MySQL** : Stockage des réservations
- **FontAwesome** : Icônes modernes
- **Git / GitHub** : Gestion du code source
- **Infomaniak** : Hébergement web

---

## ⚙️ Fonctionnalités

- **Affichage des menus**

- **Formulaire de réservation** :
    - Validation client (JavaScript)
    - Sécurisation serveur (PHP + PDO)

---

## 📦 Installation locale

1. Cloner le dépôt GitHub :

```bash
git clone https://github.com/seck024/restaurant.git
```
2. Placer le dossier dans votre serveur local (ex : `htdocs` de XAMPP/MAMP).

3. Importer la base de données (`bts_project`) via phpMyAdmin.

4. Configurer la connexion à la base de données dans `public/index.php`.

5. Lancer `localhost/restaurant` dans votre navigateur.
